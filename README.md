# Elite Stainless Steel Website Mockup

A professional manufacturing website concept designed for a stainless steel fabrication company specializing in commercial stainless steel products and custom fabrication services.

## Project Overview

This mockup demonstrates a modern industrial website built with WordPress and WooCommerce in mind, focusing on lead generation and quote requests rather than online sales.

The design is targeted toward B2B customers including:

- Builders
- Architects
- Plumbers
- Commercial Contractors
- Industrial Facilities
- Government Projects

## Features

### Homepage
- Modern industrial hero section
- Company overview
- Product categories
- Fabrication services
- Call-to-action sections

### Product Catalogue
- Commercial Urinals
- Wash Troughs
- Linear Drains
- Stainless Steel Grates
- Benchtops

### Product Pages
Each product includes:

- Product images
- Technical specifications
- Material options
- Downloadable PDF datasheets
- Quote request functionality

### Custom Product Configuration

Customers can specify:

- Length
- Width
- Depth
- Material Grade (304 or 316 Stainless Steel)
- Surface Finish
- Custom Notes

### Add To Quote System

Unlike traditional eCommerce websites, this project uses a Quote Request workflow:

1. Browse products
2. Add multiple products to quote list
3. Enter custom requirements
4. Submit one quote request
5. Receive quote via email

No online payment processing required.

## Project Goals

The objective is to provide a low-maintenance, professional manufacturing website that allows internal staff to:

- Add products
- Edit specifications
- Upload PDF sheets
- Manage enquiries
- Update images

without requiring developer assistance.
